## Dots Game - [Android Studio App]

An offline multiplayer app for users to play the lgenedary DOTS game. A user friendly UI which helps the gamers to focus on strategies and different permutations to use it at their advantage and win over their opponents.

#### How to start

Install the app first and follow these steps:
1. Follow the main menu instructions
   1. Set color for each player
   1. All set, proceed to play now

#### How to play

1. Click between any 2 adjacent dots to draw a line
1. Each player gets by default one chance per round
1. If the player creates a complete small square, then the players gets a chance to draw another line
1. Game is over once there are no lines left to draw and all the small squares are completed
1. The player with highest number of squares wins the game
